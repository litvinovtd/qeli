# Скриншоты qeli (v0.6.0)

UI клиента, отрендеренный офскрин (Avalonia headless, Skia). Вёрстка `MainWindow`
идентична на Windows (WPF) и macOS (Avalonia), поэтому эти кадры представляют оба
десктоп-клиента. Профили на снимках — демонстрационные (не реальные).

Сгенерировано: `dotnet run --project qeli-mac/QeliMac -- uishot release/screenshots`
