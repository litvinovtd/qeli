# qeli

VPN с современной криптографией (эфемерный X25519 + post-quantum ML-KEM-768,
ChaCha20-Poly1305) и настраиваемой обфускацией транспорта для стабильной работы в
сложных сетевых условиях. Один Rust-протокол-движок и клиенты под Android, Windows
и macOS, использующие общее нативное ядро через FFI.

## Состав репозитория

| Каталог | Что это |
|---|---|
| `qeli/` | Rust: протокол-ядро, сервер, realtls-FFI (`libqeli`) |
| `qeli-android/` | Клиент Android (Kotlin) |
| `qeli-win/` | Клиент Windows (WPF / .NET) |
| `qeli-mac/` | Клиент macOS (Avalonia / .NET) |
| `native-libs/` | Централизованные копии нативных ядер (+ `third-party/`) |
| `docs/` | Конфигурация, дизайн, аудиты |
| `scripts/` | Сборка/деплой/лаб-инструментарий |

## Сборка

- Сервер/ядро (Linux): `cargo build --release` в `qeli/`.
- Клиенты: см. `.github/workflows/ci.yml` (Android gradle, Win/mac `dotnet`) и
  скрипты сборки нативных ядер в `scripts/`.
- Конфигурация — `docs/CONFIG.md`.
